# Authenticity for the viva (honest and strong)

Your supervisor and examiners are **not** grading whether you typed every character from memory. They grade whether **you** understand the **design**, can **operate** the **code**, and can **defend** the **thesis** with **evidence**.

---

## What actually convinces them

1. **You explain while the IDE is open:** for each file, say **what goes in**, **what comes out**, and **one trade-off** you made (for example **kNN** because there is **no** **IP** in the feature list).  
2. **You trace numbers to files:** open **`results_table.csv`** next to **Table 7** in Word.  
3. **You show you can run or rerun:** **`run_all.py`** on a **small** set (`--nrows 10000`) if full data is slow.  
4. **Optional:** **`git log`** on **`src/`** shows **your** work over **time** in **commits**.  

---

## What to avoid

- Saying **“I never used any help”** if that is **not true**. Academic integrity rules expect **honesty**. If the programme allows certain tools, follow **that** policy in writing (handbook / spec).  
- Memorising long paragraphs **without** understanding. Short **spoken** sentences + **file** pointers are **better**.  
- Hiding **`.cursor/`** or other folders **to look** “more manual.” A clean tree is **fine**, but **honesty** is **better** than **performance**.

---

## One sentence you can use if they ask about tools

“I built this in **VS Code** with **Python**, **PyTorch**, and **PyG**, following my **design** in **Chapters 4–6**; I can **walk** **every** **main** **file** and **show** where **each** **result** **comes** from.”

(Adjust to match **your** **school’s** **exact** **rules** on **declaration** / **AI** **disclosure**.)

---

## If stress is about “does it look like my work?”

Make the **walkthrough personal**:

- Name **one** bug you **fixed** during **`graph_builder`** or **FL** wiring.  
- Name **one** **limit** you **added** in **Chapter** **9** because **you** saw **high** **F1** on a **lab** **subset**.  

That is **your** voice, not a generic **AI** **essay**.
