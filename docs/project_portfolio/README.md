## One-minute summary

- **What this is:** IoT **flow** intrusion-detection prototype on **CICIoT2023**: *k*NN **graphs** → **GAT+GRU** → optional **Flower FedAvg** → **Captum** explanations → **FastAPI** ECS-style **JSON** alerts (CPU-oriented).
- **Where the code is:** `src/`, with `config/experiment.yaml` and `scripts/run_all.py`.
- **Where the numbers are:** `results/metrics/` and `results/figures/` (regenerate after a full run).

---

## Reproduce (Python)

1. Clone the repository.
2. Create a venv and `pip install -r requirements.txt` (then install **PyTorch Geometric** for your platform).
3. Download [CICIoT2023](https://www.unb.ca/cic/datasets/iotdataset-2023.html) → place `train.csv`, `validation.csv`, `test.csv` under `data/raw/`.
4. Run: `python scripts/run_all.py --config config/experiment.yaml`
5. Details: **[`../../SETUP_AND_RUN.md`](../../SETUP_AND_RUN.md)**

`data/` is never committed; `results/checkpoints/*.pt` are **local** (regenerated by the pipeline).
