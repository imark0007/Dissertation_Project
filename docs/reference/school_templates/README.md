# School submission templates (reference copies)

These files are **UWS / programme templates** kept next to the repository for convenience. They are **not** part of the Python codebase.

- Complete and sign the forms as required by Moodle, then insert them into the **front** of the final Word/PDF (see `Arka_Talukder_Dissertation_Final_DRAFT.md` — Front Matter).
- Official wording and deadlines always follow **Moodle and the module co-ordinator**, not this folder.

| File | Typical use |
|------|-------------|
| `DissertationFrontSheet.doc` | Front sheet |
| `Declaration of originality.docx` | Signed declaration |
| `Dissertation Library Form.DOC` | Library release |
| `guideline of dissertation.docx` | Formatting / appendix rules |

**Handbook (programme-wide):** `../MSc Project Handbook 2526.docx` (same `docs/reference/` folder).

**Updated final-report wording:** repo root `Updated_Guideline For FINAL REPORT.docx` (if present).

Other checklist/example `.docx` copies may be re-added from Moodle if needed.
