# School submission templates (reference copies)

These files are **UWS / programme templates and checklists** kept next to the repository for convenience. They are **not** part of the Python codebase.

- Complete and sign the forms as required by Moodle, then insert them into the **front** of the final Word/PDF (see `Dissertation_Arka_Talukder.md` — Front Matter).
- Official wording and deadlines always follow **Moodle and the module co-ordinator**, not this folder.

| File | Typical use |
|------|-------------|
| `DissertationFrontSheet.doc` | Front sheet |
| `Declaration of originality.docx` | Signed declaration |
| `Dissertation Library Form.DOC` | Library release |
| `guideline of dissertation.docx` | Formatting / appendix rules |
| `MSc project submission checklist.docx` | Submission checklist |
| `Example of abstracts..docx` | Abstract style reference |
| `Submitting Your Dissertation.docx` | Submission process notes |
